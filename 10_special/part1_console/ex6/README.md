**Задача №3:**  
Дано трехзначное число (N1). Определить обратное число (N2). Вычислить, какое из этих двух чисел больше или они равны (вывод: 1 или 2 или 3) .   
Например: 
    
    219 ==> 912 2           724  ==> 427 1        313 ==> 313 3
___
Sample Input:
```
219
```
Sample Output:
```
912 2
```
