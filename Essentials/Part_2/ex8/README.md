Заданы две клетки шахматной доски. Если они покрашены в один цвет, то выведите слово yes, а если в разные цвета – то no.

**Входные данные**  
Вводятся 4 числа - координаты клеток.

**Выходные данные**  
Выведите ответ на задачу.
___
Sample Input 1:
```
1
1
2
2
```
Sample Output 1:
```
yes
```
___
Sample Input 2:
```
1
1
2
3
```
Sample Output 2:
```
no
```
