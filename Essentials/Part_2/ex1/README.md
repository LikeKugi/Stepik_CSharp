Дано целое положительное число - возраст человека. Если человек младше 18, то выведите строку ребёнок. Если человек младше 70, то выведите строку взрослый. В остальных случаях выведите строку старик.
___
Sample Input 1:
```
5
```
Sample Output 1:
```
ребёнок
```
___
Sample Input 2:
```
90
```
Sample Output 2:
```
старик
```
___
Sample Input 3:
```
18
```
Sample Output 3:
```
взрослый
```
