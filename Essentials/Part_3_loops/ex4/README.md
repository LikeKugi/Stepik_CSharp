Дано целое число, не меньшее 2. Выведите его наименьший натуральный делитель, отличный от 1.

**Формат входных данных**  
Вводится целое положительное число.

**Формат выходных данных**  
Выведите ответ на задачу.
___
Sample Input 1:
```
4058
```
Sample Output 1:
```
2
```
___
Sample Input 2:
```
1717
```
Sample Output 2:
```
17
```
___
Sample Input 3:
```
3311
```
Sample Output 3:
```
7
```
